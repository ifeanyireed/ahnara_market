# Ahnara Market 🛒✨

Welcome to **Ahnara Market**, the commerce application workspace of the **Ahnara Med Suite**.

Ahnara Market is a premium, full-lifecycle commerce application that serves as the last-mile retail pharmacy, lab bookings catalog, and medical device storefront for families. It is part of the Network & Commerce Layer of the Ahnara platform.

---

## 📂 Repository Structure

The project is structured as follows:

*   **📁 [web-app/](file:///Volumes/ReedBreedCC/ahnara/ahnara_market/web-app)**: The Next.js web application. Includes storefront, vendor dashboards, buyer profiles, checkout pipelines, and admin consoles.
*   **📁 [mobile-app/](file:///Volumes/ReedBreedCC/ahnara/ahnara_market/mobile-app)**: Cross-platform mobile application client (placeholder for future implementation).

## 🚀 Getting Started

### Prerequisites

Make sure you have Node.js (v18+) and npm/pnpm installed on your system.

### Running the Web App

1. Navigate to the web-app directory:
   ```bash
   cd web-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
